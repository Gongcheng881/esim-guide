## 🚀 怎么用

### 方式一：直接访问网页（推荐 ✅）

打开浏览器，输入以下链接即可：

```
https://gongcheng881.github.io/esim-guide/
```

手机、电脑、平板都能看，图片全部加载好。

### 方式二：下载到本地看

> ⚠️ **常见问题**：如果只下载了 `index.html` 一个文件就打开，图片会全部裂掉。

**正确操作**：

1. 点击本页右上角绿色的 **「Code」** 按钮 →  **「Download ZIP」**
2. 解压 ZIP 文件，得到一个文件夹（里面包含 `index.html` + `assets/` 图片文件夹）
3. **双击文件夹里的 `index.html`**，浏览器会自动打开

> 💡 一定要保持 `index.html` 和 `assets/` 文件夹在同一个目录里，图片才能正常显示。
